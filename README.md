# AI Text to Image Generator

This project is an AI/ML based Text to Image Generator developed in Python
using Stable Diffusion.

## Features
- Converts text prompts into realistic images
- Supports long prompts
- Fast image generation using GPU
- Automatically saves generated images
- Download option for generated images

## Technologies Used
- Python
- Stable Diffusion (Diffusion Model)
- Gradio
- Deep Learning
